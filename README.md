# Sunny's Dalamud Plugin Repository

Custom Dalamud plugin repository for all of Sunny's FFXIV plugins.

⚠️ **Currently in Testing** — All plugins are early development versions. Use at your own risk!

## Installation

1. Type `/xlsettings` in FFXIV chat
2. Go to: **Experimental** → **Custom Plugin Repositories**
3. Add this link:
   ```
   https://raw.githubusercontent.com/SunnysTTV/sunnysrepo/main/
   ```
4. Click **+** and **Save**
5. Open **Available Plugins** in Dalamud Plugin Installer
6. Search and install available plugins!

## Available Plugins

- **VenueMapper** ⚠️ **Testing** — FFXIV housing venue discovery with maps, events, and owner tools
